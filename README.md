# 📚 My Bookshelf App

My Bookshelf App is an **Android book tracking application** that allows users to manage books across different reading stages. Users can browse books, add them to their personal library, mark them as **To Be Read (TBR)** or **Read**, and view detailed information for each book.

---

## ✨ Features

- 📖 Browse a list of books
- ⭐ Mark books as **To Be Read (TBR)** or **Read**
- 🗑 Delete books from the library
- 📋 View detailed information about each book
- 📱 Bottom navigation to switch between book categories
- 🔄 Full **CRUD operations** using SQLite
- 🧪 Basic **UI testing with Espresso**

## 📱 App Structure

Main components of the application include:

- **MainActivity** – Entry point of the app and database initialization  
- **ListActivity** – Displays all available books  
- **TbrActivity** – Shows books marked as *To Be Read*  
- **ReadActivity** – Shows books marked as *Read*  
- **BookInfoActivity** – Displays detailed information for a selected book  
- **DBHelper** – Handles SQLite database operations  
- **RecyclerViewAdapter** – Displays books in a scrollable list  
- **Book Data Class** – Represents book information

---

## 📂 Key Functionality

- **RecyclerView** for displaying books in a dynamic list  
- **SQLite database** for persistent local storage  
- **Bottom Navigation Fragment** for easy navigation between sections  
- **Intent-based navigation** between activities  
- **Status-based filtering** (List / TBR / Read)
